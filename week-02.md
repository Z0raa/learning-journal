# Week 02 - Binary Analysis

## Tujuan Pembelajaran

Mempelajari dasar-dasar Binary Analysis sebagai langkah awal dalam Reverse Engineering.

## Materi yang Dipelajari

- Binary File
- Executable
- Struktur file executable
- Informasi dasar sebuah binary

## Tools

- Ghidra

## Hal yang Saya Pelajari

Saya belajar bahwa sebelum melakukan analisis terhadap suatu program, kita perlu mengetahui jenis file yang akan dianalisis. Informasi seperti arsitektur, format executable, serta compiler dapat membantu memahami karakteristik program tersebut.

Pada pertemuan ini saya mulai mengenal antarmuka Ghidra sebagai tools Reverse Engineering.

## Kendala

Masih beradaptasi dengan tampilan Ghidra karena memiliki banyak menu dan fitur.

## Kesimpulan

Binary Analysis merupakan tahap awal yang penting sebelum melakukan analisis lebih lanjut.
