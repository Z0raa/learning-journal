# Week 04 - Static Analysis

## Tujuan Pembelajaran

Mempelajari teknik Static Analysis terhadap file executable.

## Materi yang Dipelajari

- Analisis tanpa menjalankan program
- Header File
- Strings
- Import Library

## Tools

- Ghidra

## Hal yang Saya Pelajari

Saya belajar bahwa Static Analysis memungkinkan kita memperoleh banyak informasi mengenai sebuah program tanpa harus menjalankannya. Informasi tersebut dapat berupa struktur file, string, maupun fungsi yang digunakan.

## Kendala

Masih memerlukan waktu untuk memahami struktur executable.

## Kesimpulan

Static Analysis menjadi langkah awal yang efektif sebelum melakukan Dynamic Analysis.
