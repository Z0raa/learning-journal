# Week 05 - Dynamic Analysis

## Tujuan Pembelajaran

Memahami proses Dynamic Analysis pada executable.

## Materi yang Dipelajari

- Dynamic Analysis
- Monitoring perilaku program
- Pengamatan aktivitas program

## Tools

- Virtual Machine

## Hal yang Saya Pelajari

Saya memahami bahwa Dynamic Analysis dilakukan dengan menjalankan program di lingkungan yang aman untuk melihat perilaku sebenarnya.

## Kendala

Harus berhati-hati saat menjalankan sampel agar tidak mengganggu sistem utama.

## Kesimpulan

Dynamic Analysis melengkapi hasil Static Analysis karena memperlihatkan perilaku program saat dijalankan.
