# Week 07 - Analisis Malware Mirai

## Tujuan Pembelajaran

Menganalisis malware Mirai menggunakan teknik Static Analysis.

## Materi yang Dipelajari

- ELF Binary
- String Analysis
- Import Analysis
- Function Analysis

## Tools

- Ghidra

## Hal yang Saya Pelajari

Pada tahap ini saya mempelajari bagaimana malware Mirai dianalisis melalui struktur ELF, string yang terdapat pada binary, serta fungsi-fungsi yang digunakan. Saya juga mulai memahami proses identifikasi perilaku malware berdasarkan hasil analisis.

## Kendala

Jumlah fungsi yang cukup banyak membuat proses analisis membutuhkan ketelitian.

## Kesimpulan

Analisis malware membutuhkan kombinasi pemahaman teori dan kemampuan menggunakan tools.
