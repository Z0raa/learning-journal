# Week 01 - Pengenalan Reverse Engineering

## Tujuan Pembelajaran

Pada pertemuan pertama saya mempelajari konsep dasar Reverse Engineering serta ruang lingkup penggunaannya dalam dunia keamanan siber.

## Materi yang Dipelajari

- Pengertian Reverse Engineering
- Tujuan Reverse Engineering
- Static Analysis
- Dynamic Analysis
- Contoh penerapan Reverse Engineering

## Hal yang Saya Pelajari

Saya memahami bahwa Reverse Engineering adalah proses menganalisis suatu program tanpa memiliki source code. Teknik ini dapat digunakan untuk memahami cara kerja aplikasi, menemukan kerentanan keamanan, melakukan analisis malware, maupun mempelajari software lama.

Saya juga mulai memahami perbedaan antara Static Analysis dan Dynamic Analysis.

## Kendala

Masih cukup sulit memahami istilah-istilah baru seperti binary, assembly, dan executable.

## Kesimpulan

Pertemuan pertama memberikan dasar yang penting sebelum mempelajari teknik Reverse Engineering secara lebih mendalam.
